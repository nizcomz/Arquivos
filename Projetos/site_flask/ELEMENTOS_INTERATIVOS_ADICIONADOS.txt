6. COMO USAR OS NOVOS ELEMENTOS
================================================================================

Contador:
1. Clique em "➕ Incrementar" para aumentar
2. Clique em "➖ Decrementar" para diminuir
3. Clique em "🔄 Resetar" para zerar

Calculadora:
1. Digite um número no primeiro campo
2. Selecione a operação desejada
3. Digite outro número
4. Clique "Calcular"
5. O resultado aparece com animação

Formulário:
1. Preencha Nome, Email e Mensagem
2. Clique "Enviar Mensagem"
3. Receba confirmação com timestamp
4. Formulário é limpo automaticamente

Tema:
1. Clique no botão "🌙 Tema Escuro" / "☀️ Tema Claro"
2. O tema muda com transição suave
3. Sua preferência é salva no navegador

Cores:
1. Clique no seletor de cor ou use "Cor Aleatória"
2. Clique "Aplicar Cor" para mudar o fundo
3. Cor aleatória escolhe entre 10 opções predefinidas

Informações:
1. Visualize hora e data atualizadas em tempo real
2. Veja informações do seu navegador
3. Tema ativo está sempre visível

9. COMUNICAÇÃO AJAX
================================================================================

Fluxo de Dados:

Calculadora:
Frontend (JavaScript) → POST /api/calcular → Backend (Python)
                     ← JSON Response ← 

Formulário:
Frontend (JavaScript) → POST /api/processar-formulario → Backend (Python)
                      ← JSON Response (sucesso/erro) ←

Ambas utilizam Fetch API (moderna, sem jQuery necessário)
